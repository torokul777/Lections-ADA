git init - создает git репозиторий из текущей папки (пишется один раз в самом начале)
git add . (либо вместо точки любой файл из дериктории)- 
добавляет файлы и папки для отслеживание в index
(эту команду пишем каждый раз перед командой commit)
git commit -m 'сообщение' - создает новую версию если были изменнения в остлеживающихся файлах
