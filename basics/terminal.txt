Директория - это обычная папка

ctr + lt + t = Команда для запуска терминала (Ubuntu)
sudo = это команда для предоставления прав суперпользователя

ls - команда для вывода всех файлов/папок в текущей директории
ls -a = команда которая показывает все скрытые файлы/папки
ls -l - команда которая выводит более детальную информацию о файлах/папках

cd test (change directory) - команда для перехода в другую папку
cd - команда для перехода в домашнюю папку
cd .. - переходит на 1 папку выше(назад)
cd ../.. = переходит на 2 папки назад

mkdir test - команда для создания директории
mkdir test/test2 = создаст папку, в существующей папке
mkdir test test2 = создает 2 папки в текущей директории

touch test.txt - команда для создания файла

rm test.py - команда для удаления файла
rm -rf файл или директория - команда для удаления файла/директории
rmdir <название папки> - команда для удаления пустой папки

clear/ctr + l = команда/горячая клавиша для очистки терминала

mv test1.py test/ - перемещает файл test1.py в папку test (обязательно указываем / в конце названия папки )
mv LLECTIONS ADA LECTIONS ADA - переименовывает файл <LLECTIONS ADA> в <LECTIONS ADA>
mv test/ test2/ - переименовывает папку test/ на test2/

history - показывает все команды которые были использованы в этом терминале
history 10 - показывает последние 10 команд
history | grep <какое-то слово> - показывает все команды с использование слова в скобках
history | grep test

pwd (print working directory)- показывает абсолютный путь до 
текущей директории (начиная с корневой папки /)


cat <название файла> ( например: cat test.txt )- показывает содержимое файла

nano test.txt - открывает файл test.txt в редакторе nano, если такого
файла нет, то создаст его (в случае если файл не пустой)
сохранение файла, и выход из редактора nano
ctrl + x, y, ENTER
ctrl + s, ctrl + x

code test(code test.txt) - открывает в редакторе кода vs - code
указанную папку, или файл
code . - открывает vs code в текущей дериктории

apt - системный установщик пакетов
snap - установщик пакетов( но чаще всего установщик приложений)
